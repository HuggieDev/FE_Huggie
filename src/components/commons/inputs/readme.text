프로젝트 내에서 사용될 기본 input입니다
사용되는 인자로는 title, titleStyles, errorMessage, isSearch가 있습니다
title - 입력창 위의 제목이 있을 경우 작성하며 없을 경우에는 제목이 없는 입력창이 됩니다
titleStyles - 제목의 스타일을 넣어주시면 됩니다
errorMessage - 입력값에 대한 오류를 전달해주시면 됩니다
isSearch - 왼쪽에 돋보기 모양을 보여줍니다 검색의 경우 사용됩니다

검색의 경우 클릭 이벤트가 들어갈텐데 onClick 에 대한 로직은 결정 된 후 추가하겠습니다
